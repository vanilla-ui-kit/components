Vanilla UI Kit

Single-file, zero-dependency UI components for vanilla JavaScript. Drop in one script tag — no build step, no framework. Beautiful by default, automatic light/dark, fully keyboard-accessible, headless when you want.

GitHub

  Toast.success('Saved')
</script>

The contract

Every component keeps the same promises.

One file eachUMD, ES5, SSR-safe. CDN, npm, or copy-paste.
Zero dependenciesNo framework, no build step, nothing else to load.
Headless when you wantstyles: false injects nothing; bring your own CSS.
Converging, not coupledWith the optional core: one theme engine, one config call.
AccessibleWAI-ARIA patterns, full keyboard support, reduced motion.
IsolatedA salt namespace keeps host-page CSS from breaking things.
